# handoff: BrainAtom tool support extension

> from bhrain to rhachet — **COMPLETED**

## status: delivered

rhachet has released tool support. the handoff is complete.

## context

bhrain's BrainArch1 is an agentic brain that orchestrates:
1. LLM inference
2. tool execution
3. loop until done

bhrain CONFORMS to rhachet. no custom atom plugins.

rhachet's `genContextBrain({ choice: { repl: 'bhrain/arch1@claude/sonnet' } })` discovers bhrain's repl and composes it with the atom.

## the gap (closed)

| BrainArch1 needs | rhachet now provides |
|------------------|----------------------|
| `messages[]` with roles | `on.episode` continuation |
| `tools[]` definitions | `plugs.tools` ✓ |
| `toolCalls[]` in response | `calls.tools` ✓ |

## delivered contract

```ts
// BrainAtom.ask() signature
ask<TOutput, TPlugs extends BrainPlugs>(input: {
  on?: { episode: BrainEpisode };
  plugs?: TPlugs;
  role: { briefs?: Artifact[] };
  prompt: string;
  schema: { output: z.Schema<TOutput> };
}) => Promise<BrainOutput<TOutput, 'atom', TPlugs>>

// BrainPlugs - tools go in plugs
interface BrainPlugs {
  tools?: BrainPlugToolDefinition[];
}

// BrainOutput - calls come out (typed based on TPlugs)
interface BrainOutput<TOutput, TGrain, TPlugs> {
  output: TOutput | null;  // null when brain defers to tool calls
  calls: { tools: BrainPlugToolInvocation[] } | null;
  metrics: BrainOutputMetrics;
  episode: BrainEpisode;
}

// tool definition
interface BrainPlugToolDefinition {
  slug: string;
  description: string;
  schema: { input: z.Schema<unknown> };
}

// tool invocation
interface BrainPlugToolInvocation {
  exid: string;   // provider-assigned ID for correlation
  slug: string;
  input: unknown;
}
```

## acceptance criteria (met)

1. input accepts `plugs?: { tools?: BrainPlugToolDefinition[] }` ✓
2. BrainOutput includes `calls: { tools: BrainPlugToolInvocation[] } | null` ✓
3. episode continuation handles tool flow via `on.episode` ✓
