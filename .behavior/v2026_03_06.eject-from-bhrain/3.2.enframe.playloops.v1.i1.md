|------------|-------------|--------------|------------------|---------------|

## feasibility proof

### tools available

- jest: test runner with unit/integration/acceptance configs
- test-fns: given/when/then BDD helpers
- typescript: type verification via `npm run test:types`
- api keys: available via `use.apikeys.sh` skill

### setup required

1. run ejection command (rsync from worktree)
2. add absent dependencies (as-procedure, fast-glob, openai)
3. update package.json identity (name, description)
4. run `pnpm install`

### playloop code sketch

```ts
// playloop.acceptance.test.ts
import { given, when, then, useBeforeAll } from 'test-fns';

describe('BrainArch1 ejection', () => {
  given('[case1] package is installed', () => {
    when('[t0] import is attempted', () => {
      then('invokeBrainArch1 is exported', () => {
        const { invokeBrainArch1 } = require('rhachet-brains-bhrain');
        expect(invokeBrainArch1).toBeDefined();
      });
      then('BrainArch1Actor is exported', () => {
        const { BrainArch1Actor } = require('rhachet-brains-bhrain');
        expect(BrainArch1Actor).toBeDefined();
      });
    });
  });

  given('[case2] actor is configured', () => {
    const actor = useBeforeAll(async () => {
      const { BrainArch1Actor, genAtomAnthropic } = require('rhachet-brains-bhrain');
      return new BrainArch1Actor({
        atom: genAtomAnthropic({ model: 'claude-3-5-haiku-latest' }),
        role: { systemPrompt: 'you are a test assistant' },
        toolboxes: [],
        constraints: { maxIterations: 1, maxTokens: 1024 },
        memory: null,
        permission: null,
      });
    });

    when('[t1] brain is invoked', () => {
      then('result is returned', async () => {
        const { invokeBrainArch1 } = require('rhachet-brains-bhrain');
        const result = await invokeBrainArch1(
          { actor, userInput: 'say hello' },
          { log: console },
        );
        expect(result.finalResponse).toBeDefined();
        expect(result.terminationReason).toBeDefined();
      });
    });
  });
});
```
