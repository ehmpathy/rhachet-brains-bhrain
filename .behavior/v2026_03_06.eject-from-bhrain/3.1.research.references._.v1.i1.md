## ref.1 = rhachet BrainRepl interface (the contract)

**source**: `node_modules/rhachet/dist/domain.objects/BrainRepl.d.ts`

**establishes**:
- the common contract that all brain.repl implementations must conform to
- BrainRepl is a DomainEntity with unique keys: ['repo', 'slug']

**interface**:
```ts
interface BrainRepl {
  repo: string;           // e.g., "anthropic", "bhrain"
  slug: string;           // e.g., "claude-code", "bhrain/arch1"
  description: string;
  spec: BrainSpec;

  ask: <TOutput>(input: {
    on?: PickOne<{ episode: BrainEpisode; series: BrainSeries }>;
    plugs?: BrainReplPlugs;
    role: { briefs?: Artifact<typeof GitFile>[] };
    prompt: string;
    schema: { output: z.Schema<TOutput> };
  }) => Promise<BrainOutput<TOutput, 'repl'>>;

  act: <TOutput>(input: {
    on?: PickOne<{ episode: BrainEpisode; series: BrainSeries }>;
    plugs?: BrainReplPlugs;
    role: { briefs?: Artifact<typeof GitFile>[] };
    prompt: string;
    schema: { output: z.Schema<TOutput> };
  }) => Promise<BrainOutput<TOutput, 'repl'>>;
}
```

**key insight**:
> BrainRepl.ask = readonly analysis (no write tools)
> BrainRepl.act = read+write actions (full tool access)

- ask mode: disallowedTools = ['Edit', 'Write', 'Bash', 'NotebookEdit']

**pattern**:
```ts
export const genBrainRepl = (input: { slug: AnthropicBrainReplSlug }): BrainRepl => {
  return new BrainRepl({
    repo: 'anthropic',
    slug: input.slug,
    spec: config.spec,
    ask: async (askInput) => invokeQuery({ mode: 'ask', ...askInput }),
    act: async (actInput) => invokeQuery({ mode: 'act', ...actInput }),
  });
};
```

## ref.4 = relationship: BrainRepl wraps BrainArch1

**establishes**:
- BrainRepl is the PUBLIC CONTRACT (rhachet interface)
- BrainArch1 is the INTERNAL IMPLEMENTATION
- genBrainRepl must create a BrainRepl that dispatches to BrainArch1

**architecture**:
```
┌─────────────────────────────────────────────────────────┐
│ rhachet consumer                                        │
│   brain.ask({ prompt, schema, role })                   │
│   brain.act({ prompt, schema, role })                   │
└────────────────────────┬────────────────────────────────┘
                         │ conforms to BrainRepl interface
                         ▼
┌─────────────────────────────────────────────────────────┐
│ genBrainRepl({ slug: 'bhrain/arch1' })                  │
│   ├─ translates ask/act to BrainArch1 semantics        │
│   ├─ creates BrainArch1Repl with BrainAtom             │
│   ├─ invokes invokeBrainArch1                          │
│   └─ transforms BrainArch1LoopResult to BrainOutput    │
└────────────────────────┬────────────────────────────────┘
                         │ internal implementation
                         ▼
┌─────────────────────────────────────────────────────────┐
│ invokeBrainArch1({ repl, userInput }, context)          │
│   ├─ validates input                                    │
│   ├─ merges toolboxes                                   │
│   ├─ runs agentic loop via runBrainArch1Loop           │
│   │     └─ calls repl.atom.ask() for LLM inference     │
│   └─ returns BrainArch1LoopResult                      │
└─────────────────────────────────────────────────────────┘
                         │
                         │ BrainAtom from rhachet-brains-* packages
                         ▼
┌─────────────────────────────────────────────────────────┐
│ rhachet-brains-anthropic / rhachet-brains-openai        │
│   genBrainAtom({ slug }) → BrainAtom                    │
└─────────────────────────────────────────────────────────┘
```

## ref.8 = rhachet brain discovery pattern

**source**: `rhachet/src/domain.operations/brains/getAvailableBrains.ts`

**establishes**:
- brains are discovered via peer dependencies (rhachet-brains-* packages)
- discovery scans package.json for rhachet-brains-* dependencies
- each brain package exports `getBrainAtomsBy*` and `getBrainReplsBy*` functions
- callers use `genContextBrain({ choice: slug })` to select a brain

**pattern**:
```ts
// discovery (in caller's code)
const brain = await genContextBrain({ choice: 'claude/sonnet' });

// usage (passes atom via context)
const repl = genBrainRepl({ slug: 'bhrain/arch1' });
await repl.ask({ prompt, schema, role }, { brain });
```

**key insight**:
> no direct dependencies on brain suppliers — only peer dependencies
> callers choose which atom to use via slug
> this enables any rhachet-brains-* package to power BrainArch1
