## gaps found

### gap 1: no rollback plan (FIXED)

**observation:** blueprint had no documented rollback procedure if ejection fails mid-way.

**risks table mentions:**
- import path breakage → mitigation: sedreplace plan mode first
- absent dependencies → mitigation: compare package.json
- test failures → mitigation: run tests incrementally

**was absent:** what to do if:
- ejection command fails mid-file
- imports are broken and can't be fixed
- critical files are absent

**action taken:** added rollback plan section to blueprint with:
```bash
# partial ejection failure
git checkout -- .
git reset --hard HEAD~1

# import fixup failure
git checkout -- src/

# corrupted state (last resort)
rm -rf rhachet-brains-bhrain && git clone ...
```
