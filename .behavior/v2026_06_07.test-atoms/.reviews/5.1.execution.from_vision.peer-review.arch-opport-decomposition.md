# blocker.1: Duplicated test harness logic across multiple AI provider tests

**rule**: .agent/repo=bhuild/role=behaver/briefs/practices/behavior.execution/architect/rule.prefer.decomposable-architecture.md

**locations**:
- src/domain.operations/arch1/core/invokeBrainArch1.fireworksai.deepseek-v4-flash.codewrite.integration.test.ts
- src/domain.operations/arch1/core/invokeBrainArch1.fireworksai.m25.codewrite.integration.test.ts
- src/domain.operations/arch1/core/invokeBrainArch1.fireworksai.m27.codewrite.integration.test.ts
- src/domain.operations/arch1/core/invokeBrainArch1.xai.codewrite.integration.test.ts

The entire test body for code generation (network sandbox setup with curl wrapper, creation of 'test' and 'chmod' tools via genBrainPlugToolDeclaration, BrainArch1Config construction, invokeBrainArch1 call, file verification, and post-run assertions) is duplicated almost verbatim across four test files. This violates the core requirement for decomposable architecture: code must decompose into reusable operations (transformers for compute, communicators for i/o, orchestrators for composition). Instead of extracting shared setup into composable leaf operations (e.g., setupNetworkSandbox, createVerificationToolbox, runCodegenAndVerify), each file repeats the full inline logic. This prevents recomposition, makes evolution expensive (any change to the harness must be replicated in all copies), and turns the test 'orchestrator' into unreadable machine code rather than narrative. With 4+ near-identical usages, extraction is required per the rule's guidance on grain clarity, orchestrator narrative, and recomposition potential.

**snippet**:
```ts
// curl wrapper: logs calls, blocks non-allowlisted domains
const curlWrapper = `#!/usr/bin/env bash
ALLOWED_DOMAIN="api.open-meteo.com"
LOG_FILE="${networkLogPath}"

# log the call
echo "CURL_CALL: $*" >> "$LOG_FILE"

# find url argument
URL_ARG=""
for arg in "$@"; do
  if [[ "$arg" == http* ]]; then
    URL_ARG="$arg"
    break
  fi
done

# block unauthorized domains
if [[ -n "$URL_ARG" && "$URL_ARG" != *"$ALLOWED_DOMAIN"* ]]; then
  echo "BLOCKED: $URL_ARG" >> "$LOG_FILE"
  echo "Error: Network call to unauthorized domain blocked: $URL_ARG" >&2
  exit 1
fi

# pass through to real curl
${realCurlPath} "$@"
`;
await fs.writeFile(path.join(binDir, 'curl'), curlWrapper);
await fs.chmod(path.join(binDir, 'curl'), 0o755);
```
